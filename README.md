# DTracker - Trucking management system

- Goal: Provide a simple way for brokers in the trucking industry to track and manage drivers assigned to truck tours for delivery across the globe.

- Inspiration: A very close friend is working as a trucking broker in an US trucking company, so after quite a lot of talks about his job and how he is working we've came to a conclusion that they are using a very outdated and pricey software for their job (tracking and management). Just because of that I've got the inspiration to create something similar and also that I can use this as my diploma thesis project for defense.

- Motivation: In the era of AI and everyone forcing AI to be used everywhere for everything I wanted to have a hobby project where I can do raw programming without using AI for any code generation, rather than using AI for learning and explaining concepts. Also I wanted to learn and repeat some core backend/frontend concepts and technologies more in depth, so that's why I went with this approach so I can read more documentation for the technologies/concepts and learn them in more depth.
